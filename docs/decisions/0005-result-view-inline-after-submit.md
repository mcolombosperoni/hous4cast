# 0005 - Result view inline after submit

- Status: accepted
- Date: 2026-04-25

## Context
The legacy reference shows a second page result, but UX should be modern and smoother.

## Decision
Show the estimation result inline as the final step after submit.

## Alternatives considered
- Separate result page: postponed and not needed for MVP.

## Consequences
- Better continuity and lower friction on mobile.
- Simpler state handling in the SPA.
